## О проекте

**Intelligent Math Solver** — это веб-приложение для автоматизации решения математических задач численными методами. Система предназначена для использования в учебной и научной деятельности кафедры прикладной математики КубГУ.

### Основные возможности:
- решение нелинейных уравнений;
- численное интегрирование;
- решение обыкновенных дифференциальных уравнений (ОДУ);
- решение систем линейных алгебраических уравнений (СЛАУ);
- интеллектуальная обработка запросов на естественном языке (нейросетевая подсистема);
- визуализация процессов решения (графики, таблицы итераций);
- накопление истории решений для обучения нейросети.

---

## Функциональные возможности

| Класс задач | Реализованные методы |
|-------------|----------------------|
| Нелинейные уравнения | Ньютона, половинного деления, секущих, простой итерации |
| Численное интегрирование | Симпсона, трапеций, прямоугольников (средних), Монте-Карло |
| Обыкновенные ДУ | Эйлера, Рунге-Кутты 4-го порядка |
| СЛАУ | Гаусса, Якоби, Зейделя |
| Интеллектуальный анализ | Нейросетевая подсистема на TensorFlow.js (распознавание типов задач, рекомендация методов) |

## Установка и запуск

### Требования
- Python 3.10+
- PostgreSQL 14+
- Современный браузер (Chrome, Firefox, Edge)

### Инструкция

1. **Клонировать репозиторий**
```bash
   git clone https://github.com/Vikkkaaa123/intelligent-math-solver.git
   cd intelligent-math-solver
```
2. **Настроить базу данных**
```bash
   python init_db.py
```
3. **Запустить сервер FastAPI**
```bash
python app.py
```
4. **Открыть клиент**

- Локально: откройте файл `index.html` в браузере
- Через сервер: перейдите по адресу `http://localhost:8000`

> Внимание: при отсутствии соединения с сервером вычисления будут работать, но сохранение результатов в базу данных станет недоступно.
